Projeto Controle de Clientes

API de Serviços baseados em REST (WebApi)
  • POST - /api/cliente/cadastro
  • Descrição: Cadastrar um novo cliente
  • Tipo de dados: JSON
  • Dados de entrada:
    • Nome - Texto
    • Email - Texto
  • Saída HTTP:
    • 200 - OK (Cliente cadastrado com sucesso)
    • 403 - Erros de validação dos dados
    • 400 - Erro do servidor
  • PUT - /api/cliente/editar
  • Descrição: Editar dados de um cliente
  • Tipo de dados: JSON
  • Dados de entrada:
    • ClienteID - Inteiro
    • Nome - Texto
    • Email - Texto
  • Saída HTTP:
    • 200 - OK (Cliente atualizado com sucesso)
    • 403 - Erros de validação dos dados / cliente não encontrado
    • 400 - Erro do servidor
  • DELETE - /api/cliente/excluir
  • Descrição: Excluir um cliente
  • Tipo de dados: JSON
  • Dados de entrada:
    • ClienteID - Inteiro
  • Saída HTTP:
    • 200 - OK (Cliente excluido com sucesso)
    • 403 - Cliente não encontrado
    • 400 - Erro do servidor
  • GET - /api/cliente/obter
  • Descrição: Obter dados de 1 Cliente
  • Tipo de dados: JSON
  • Dados de entrada:
    • ClienteID - Inteiro
  • Saída HTTP:
    • 200 - OK (Objeto contendo os dados do Cliente)
    • 403 - Cliente não encontrado
    • 400 - Erro do servidor
  • GET - /api/cliente/listar
  • Descrição: Listar todos os Clientes
  • Tipo de dados: JSON
  • Saída HTTP:
    • 200 - OK (Lista de objetos contendo os dados dos Clientes)
    • 403 - Nenhum Cliente encontrado
    • 400 - Erro do servidor
